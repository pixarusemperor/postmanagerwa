# T-005: API Route Contract ‚Äî PostManager Routes (/api/pm/*)

`wayfinder:task` | Parent: map.md

## Question

Produce the complete API route specification for all PostManager REST endpoints. Every route must define: HTTP method, path, auth requirements, request body schema, response body schema, error responses, and which tables it touches.

## Requirements

1. Every route is scoped to the requesting user's organization (via JWT org_id claim)
2. Every request/response body has a JSON Schema or TypeScript type definition
3. Every error response follows a consistent format: `{ error: string, code: string, details?: object }`
4. Pagination uses cursor-based pagination where lists can be large (dispatches, action logs, products)
5. File upload routes use multipart/form-data with size limits
6. Auth middleware spec: JWT extraction, org_id scoping, role checking
7. Rate limiting strategy defined per route group
