# T-011: Infrastructure & Deployment Specification

`wayfinder:task` | Parent: map.md

## Question

Produce the infrastructure specification: Cloudflare R2 bucket setup, Supabase project configuration, deployment target, environment variables, CI/CD pipeline, and monitoring.

## Scope

### Cloudflare R2
1. Bucket naming and structure (`postmanagerwa-media/{organization_id}/{product_id}/...`)
2. CORS configuration for presigned uploads from the Next.js domain
3. Custom domain setup (`assets.postmanagerwa.com` → R2 bucket)
4. Image transformation rules (thumbnail URLs via query params)
5. Lifecycle policies (auto-delete export ZIPs after 7 days, move raw imports to IA after 30 days)
6. Access key rotation policy

### Supabase
1. Project configuration (region, compute size, connection pool settings)
2. Database extensions to enable (`pgcrypto`, `ltree`, `pg_cron`, `pg_partman`)
3. Auth settings (email templates, redirect URLs, allowed OAuth providers)
4. Storage: disable Supabase Storage (use R2 instead) or keep for small assets
5. Realtime: enable for dispatch queue updates and notification delivery
6. Backup strategy (Point-in-Time Recovery, daily backups, retention)

### Deployment
1. Target: Coolify on VPS (same as WassFlow: `coolifyone.orizongroup.online`) or Vercel
2. Domain: `app.postmanagerwa.com` (or subdomain of choice)
3. Dockerfile spec (Next.js standalone output, health check, env vars)
4. Environment variables specification (all env vars with descriptions, which are secret, which are per-environment)
5. SSL/TLS via Traefik (Coolify) or Vercel's built-in

### CI/CD
1. GitHub Actions workflow: lint → typecheck → test → build → deploy
2. Preview deployments for PRs
3. Database migration CI (run migrations against staging before production)
4. Secret management (GitHub Secrets → Coolify env vars)

### Monitoring
1. Health check endpoint (`GET /api/health`)
2. Error tracking (Sentry or similar)
3. Performance monitoring (Vercel Analytics or custom)
4. Uptime monitoring
5. Cost monitoring (R2 storage, Supabase compute, AI tokens)
